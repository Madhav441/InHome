# Sentinel AU web dashboard

Next.js dashboard for guardians to manage policies, schedules, alerts, and privacy controls.

## Development

```bash
pnpm install
pnpm --filter @sentinel-au/web dev
```

## Features

- Synthetic policy matrix and heatmap to illustrate data visualisation.
- Privacy centre copy aligned to APPs.
- Pairing workflow showing QR codes and expiring tokens.
