# Sentinel AU iOS agent

SwiftUI app leveraging FamilyControls, DeviceActivity, and ManagedSettings to apply schedules without private entitlements.

## Highlights

- Requests FamilyControls authorisation on launch.
- Applies a demo school-hours schedule and limits social networking categories.
- Designed for TestFlight distribution under consumer rules.

## Development

Open `Package.swift` in Xcode 15 and run on an iOS 17+ device.
